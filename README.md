## Tensorflow 기본 사용법
- tf.random_normal
> 정규분포(normal distribution) - 통계 확률분포에서 가장 중요한 분포로 종 모양(bell shpae)으로 표현되는 확률
- tf.random.uniform
>균등(균일)분포(uniform distribution) - 각각의 구간에서 동일한 확률로 표현되는 분포로 사각형 모양
